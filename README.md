## A simple layer7-dstat run with Golang and HTML
### TO Build 
```sh
git pull https://github.com/IsMio/layer7-dstat-go
go build main.go
```
### Usage 
```sh
main -a[Web server address] -n[NIC number]
main -l --show all NICs
```
